Animation work lets you give a swapped model new movement, or add custom animations to a Pal or player
without breaking the animations the game already relies on.

## In this section

- **[Custom animations](/animations/custom-animations/)** - how to create custom animations without
  breaking all the others.

Animation modding sits on top of the [asset pipeline](/asset-modding/) - you'll usually be working with
skeletons and models you extracted with the [tools](/tools-and-setup/).
