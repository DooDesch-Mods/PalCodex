Modding a dedicated server has its own rules: some mods are client-side only, some must be installed
server-side, and administration is often done with a dedicated admin toolkit.

## In this section

- **[Server modding](/dedicated-server/server-modding/)** - how mods are installed and loaded on a
  Palworld dedicated server, and what "server-side" means for a mod.

For server administration - giving items, managing players, anti-cheat - see
[PalDefender](/frameworks/paldefender/). Compatibility after a game patch is covered in
[Compatibility & updates](/troubleshooting/compatibility-and-updates/).
