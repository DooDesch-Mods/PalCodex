Asset modding changes what the game *looks* like: swapping a Pal or player model, retexturing an
existing one, or adding attachments and physics. It works by editing cooked assets and repacking them.

## In this section

- **[Cooked uasset basics](/asset-modding/uasset-basics/)** - what a cooked `.uasset` is and how its
  pieces fit together, so edits make sense.
- **[Model swaps](/asset-modding/model-swaps/)** - replace one model with another.
- **[Texture swaps](/asset-modding/texture-swaps/)** - retexture without Blender or Unreal Engine.
- **[Kawaii physics](/asset-modding/kawaii-physics/)** - add jiggle/bone physics to player model swaps.

Once you've made an asset change, package it as described in
[Packaging & Pak Pipeline](/packaging/). For the tools, see [Tools & Setup](/tools-and-setup/).
